### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jayson-s/life_balance_plus.git
   cd life_balance_plus
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   - For Android:  
     ```bash
     flutter run -d android
     ```
   - For iOS:  
     ```bash
     flutter run -d ios
     ```
   - For Web:  
     ```bash
     flutter run -d chrome
     ```
